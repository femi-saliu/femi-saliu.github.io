Projects

Home Harmony

I contributed to the creation of this application, which is a desktop/mobile app to solve the problem of communication and delegating tasks between roommates. Our app allows users to create a virtual house and add other users as roommates. Then users can manage house expenses, delegate tasks to their roommates and post anonymous concerns/compliments that their roommates can also view. This application was created using Angular.js, Firebase, and Material Design Lite.

Park Assist

My team inherited an application whose purpose was to recommend nearby available parking spaces for drivers in Santa Monica. The application scrapes data from Santa Monica's parking meter API and uses that data to route the user to the nearest available parking meter using Google Map's API. After my group recieved the project, we added additional features. My job specifically was to extend the application to recommend parking lots in addition to metered parking spaces. Also, I implemented a feature that lets users "reserve" a space, which hides a particular space from other people using the app for 10 min. This app was created using Santa Monica's Parking API, Google Map's API, Angularjs, and Firebase.

Goblin Rush

I made Goblin rush as fun side project while I was studying programming in Budapest, Hungary. I ended up releasing the application to the iOS app store and for the year that it was on the store, over 700 people from all around the world downloaded it. I used a library called UnityScript, which is based on Javascript to program all of the ingame logic, Adobe photoshop to create all of the images and assets, and Unity3D to pull everything together.
